Cascadia Ruby 2013: Ruby Systems Programming

Tue, Oct 22, 2013

Note: This post is an un-contexualized, un-edited, & un-proofed bunch of notes taken during Cascadia Ruby 2013. See all the notes from this conference.

talk by: Andy Delcambre

The kernal

  • Runs in kernal mode.
  • Not your code.
  • Can do anything and everything.

Your Code

  • Runs in user mode.
  • Math.
  • Access memory.

System calls

  • Ask the kernal nicely to do things.
  • Api to kenal mode.
  • If it is not in the syscall_table you can not call it.
  • Unix has 261 syscalls implemented.
  • Windows has thousands listed (implemented/unimplemented).
